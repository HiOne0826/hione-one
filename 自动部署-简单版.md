# hione.one 自动部署（简单版）

## 工作原理

1. **你在本地：** 写代码 → 构建 → 提交并推送到 GitHub
2. **服务器：** 每 5 分钟自动检查 GitHub 有没有新提交 → 如果有，就自动拉取并部署

---

## 步骤 1：本地设置（已完成）

✅ 修改了 `.gitignore`，允许提交 `dist/` 目录

## 步骤 2：每次更新时执行

在本地项目目录运行：

```bash
cd /Users/ericgao/Desktop/hione-one

# 1. 构建项目
npm run build

# 2. 提交并推送
git add .
git commit -m "Update website"
git push origin main
```

## 步骤 3：服务器上设置（我来帮你）

服务器会：
- 每 5 分钟检查一次 GitHub
- 如果有新提交，自动拉取并部署到 `/var/www/hione.one/`

---

## 就这么简单！

以后你只需要：
1. 改代码
2. 运行 `npm run build`
3. `git add/commit/push`

服务器会自动更新网站！
