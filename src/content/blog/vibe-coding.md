# Vibe Coding：我是怎么用 AI 一天写完一个网站的

从 0 到 1 用 Astro + Tailwind + OpenClaw 做个人官网。

## 什么是 Vibe Coding

不是让 AI 写代码，而是和 AI 一起创作...

## 我的工作流

1. 先在脑子里想好大概的样子
2. 让 OpenClaw 生成第一版
3. 迭代、调整、再迭代

## 工具链

- Astro - 框架
- Tailwind - 样式
- OpenClaw - AI 搭档
